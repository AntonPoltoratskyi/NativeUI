# NativeUI

## Requirements:
- iOS 9.0
- Xcode 11.0
- Swift 5

## Installation

#### CocoaPods

```ruby
target 'MyApp' do
  pod 'NativeUI', '~> 1.0'
end
```

If you don't need to connect all UI components you may use subspecs like:

```ruby
target 'MyApp' do
  pod 'NativeUI/Alert', '~> 1.0'
end
```

Available subspecs:
- `Utils`
- `Alert`
